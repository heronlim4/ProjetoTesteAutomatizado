Projeto de teste simples para utilização do Selenium e JUnit na construção de testes automatizados.

Recomendo a utilização do Firefox.

Documentação do Selenium:
* https://www.selenium.dev/documentation/webdriver/getting_started/
* https://github.com/SeleniumHQ/seleniumhq.github.io

Documentação do JUnit:
* https://docs.junit.org/current/user-guide/
* https://github.com/junit-team/junit-examples/tree/r5.13.4/junit-jupiter-starter-maven

Extensões do navegador:
* Selector's Hub - https://selectorshub.com/selectorshub/
* Selenium IDE - https://addons.mozilla.org/en-US/firefox/addon/selenium-ide/
